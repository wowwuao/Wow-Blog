## 1.JS有几种模块规范？
#### 1. CommonJS
它通过 require 来引入模块，通过 module.exports 定义模块的输出窗口 **`Node`** 实现了CommonJS规范

#### 2.AMD
异步方式加载模块，推崇依赖前置，全部加载在执行 。require.js

#### 3.CMD
异步加载，就近依赖，加载一个执行一个。  sea.js

#### 4.ES6提出的方案
使用 import export 引入和导出模块
