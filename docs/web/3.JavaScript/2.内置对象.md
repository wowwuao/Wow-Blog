# 内置对象
## JS有哪些内置对象
1. 值属性

    Infinity、NaN、undefined、null
2. 函数属性

    全局函数可直接调用
    eval()、parseInt()、parseFloat()
3. 基本对象

    Object、Function、Boolean、Symbol、Error
4. 数字和日期对象

    Math、Number、Date
5. 字符串

    String、RegExp
6. 可索引的集合对象

    Array
7. 使用键的集合对象

    Map、Set、WeakMap、WeakSet
8. 矢量集合

    SIMD
9. 结构化数据
    
    JSON
10. 控制抽象对象

    Promise、Generator
11. 反射

    Reflect、Proxy
12. 国际化
    为了支持多语言而加入的对象

    Intl、Intl.Collator
13. WebAssembly
14. 其他

    argument
